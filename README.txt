This is a README as to how to function the web app
1. Ensure that flask and python are installed correctly
2. Activate flask enviornment with source 'yourFlaskEnviornmentName'/bin/activate
3. Export APIWebGo.py as your Flask App using export FLASK_APP=hello.py
4. Ensure you have backgroundCode.py in the same directory as APIWebGo
5. Run the Flask app with flask run
6. Sign in and allow APIWebGo to view your Google Calendar
